# NanamiBot
这是一个娜娜米的聊天机器人（基于nonebot2/cqhttp）

<img src="https://github.com/coderrbk/NanamiBot/blob/main/nanami.jpg" width="100px">

使用说明：部分功能需要数据库存储数据，我使用的是mysql数据库，可切换成其他的数据库

前置的准备工作：

    安装nonebot2插件

    安装cqhttp
    
    安装数据库（可选，部分功能需要）

菜单中的功能已经测试可用，（部分功能在data_source.py中注明了使用方法）
